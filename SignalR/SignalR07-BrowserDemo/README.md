# 07 浏览器 HTML 联调 Demo

## 目标

这个项目专门用于“看效果”：

- 后端：ASP.NET Core + SignalR Hub
- 前端：同项目内置静态 HTML 页面
- 运行后直接打开浏览器，就能连接、发送消息、看到广播结果

## 访问方式

启动后打开首页即可：

- `/`
- Hub 地址：`/hubs/live-chat`

## 说明

前端页面通过 CDN 引入 `@microsoft/signalr` 浏览器客户端脚本。

这样做的原因是：

- 当前仓库没有现成的本地 JS 客户端资源
- 代码仍然保持前后端同项目托管，使用方式最直接

如果后续你希望完全离线，我可以再帮你把浏览器端依赖改成仓库内置静态资源。
